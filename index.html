Scrollet

A tilt scrolling bookmarklet for Firefox and Mobile Safari.

Inspired by the tilt scrolling feature in the Instapaper iPhone app, I decided to go ahead and make a bookmarlet to enable tilt scrolling on any webpage using the JavaScript deviceorientation API and its Firefox counterpart.
